## Tips for good screenshots
- Use a clean browser window (hide bookmarks bar, extensions)
- Crop tightly to the actual product UI — no desktop background visible
- For Rifaah Dubai, a shot of the storefront homepage or a product page works well
- For Texna OMS / Smart QR, the admin dashboard or the QR redemption screen shows the product best
- Keep file sizes under ~300KB each (use tinypng.com or similar to compress) so the site stays fast

Until these files are added, the site shows a clean placeholder automatically —
nothing breaks if you skip this step, but real screenshots will make the biggest
visual difference on the homepage.
