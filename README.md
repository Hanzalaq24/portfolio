# Portfolio — Hanzala Qureshi

Single-page developer portfolio. Fully static, no build step required.

## Structure
```
index.html          → main page
assets/js/           → three.js (3D robot mascot) + gsap + ScrollTrigger (bundled locally, no CDN dependency)
assets/img/photo.jpg → profile photo
```

## Deploy
Push this folder's contents to the root of your `portfolio` repo, then enable
GitHub Pages (Settings → Pages → Deploy from branch → main → /root).

Or just open `index.html` directly in a browser — no server needed.
